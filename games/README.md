# Game data

## `keen4.jsdos` — included (Keen 4 shareware v1.4)

This bundle contains the **Commander Keen 4 shareware** episode (v1.4), which id Software released
as **freely redistributable**. It is packaged here as a `.jsdos` bundle:

```
.jsdos/dosbox.conf   DOSBox config (autoexec runs KEEN4E.EXE)
AUDIO.CK4            \
EGAGRAPH.CK4          > Keen 4 shareware data files
GAMEMAPS.CK4         /
KEEN4E.EXE           Keen 4 shareware executable
```

Source of the shareware data: https://davidgow.net/keen/4keen14.zip

## Keen 5 & 6 — NOT included (commercial)

Episodes 5 ("The Armageddon Machine") and 6 ("Aliens Ate My Babysitter") are commercial games.
**Their data files must never be committed to this repository.** Players supply their own copies
at runtime via the file picker on the site; those files are assembled into a `.jsdos` bundle in
the browser and never uploaded anywhere.

Buy them: [GOG](https://www.gog.com/game/commander_keen_complete_pack) ·
[Steam](https://store.steampowered.com/app/9180/Commander_Keen_Pack/) (both are v1.4).
